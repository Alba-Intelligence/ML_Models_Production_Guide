# Reading Paths


# Role-Based Reading Paths

This page provides customized reading paths for different user roles and
experience levels.

## 🎯 Beginner Path (2-4 hours)

**For**: New users, data scientists new to ML deployment

### Prerequisites

- Basic Python knowledge
- Understanding of machine learning concepts
- Familiarity with command line

### Learning Path

    01_Terminology_Glossary → 
    02_Getting_Started → 
    03_Concepts_and_Architecture → 
    04_End_to_End_Workflow

### Key Objectives

- Understand ML Deploy terminology (EX-01, EX-02, EX-03)
- Set up development environment
- Learn core platform concepts
- Complete a full workflow practice

### Expected Outcomes

- ☐ Can explain key platform concepts
- ☐ Can set up local development environment
- ☐ Can execute a complete ML workflow
- ☐ Understands platform architecture

## 🚀 Intermediate Path (1-2 hours)

**For**: ML engineers with deployment experience

### Prerequisites

### Learning Path

    01_Implementation_Patterns

### Key Objectives

### Expected Outcomes

- ☐ Can troubleshoot common issues

## 💡 Tips for Success

2.  **Practice Hands-On**: Work through examples, not just reading
