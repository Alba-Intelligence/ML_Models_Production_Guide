# Terminology Glossary Summary

The terminology glossary provides comprehensive, standardized definitions for key terms and concepts used throughout the ML Deploy documentation. This is a foundational document that ensures consistent understanding of platform terminology across all user interactions.

## Purpose

This document serves as the authoritative reference for ML Deploy terminology. It provides clear, concise definitions that establish a common vocabulary for all stakeholders, ensuring consistent communication and understanding across the platform.

The terminology glossary includes:

- **Foundational Concepts**: Detailed explanations of EX-01, EX-02, EX-03 vertical slices
- **Architecture Components**: Definitions of execution backends, web UI contracts, governance gates
